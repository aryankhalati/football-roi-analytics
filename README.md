 Football Talent ROI & Market Value Analytics

A data science project developed to identify undervalued football players by normalizing performance against market valuation.

 Key Features
- Statistical Modeling: Uses Poisson distributions for performance and Log-Normal distributions for market value.
- Efficiency Metrics: Implements G+A per 90 normalization to eliminate playing-time bias.
- Visual Intelligence: Includes a comprehensive dashboard with Correlation Heatmaps and ROI Leaderboards.

 Tech Stack
- Language: Python
- Libraries: Pandas, NumPy, Matplotlib, Seaborn
- Environment: Google Colab / Local Python 3.x
